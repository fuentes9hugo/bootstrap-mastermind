## ¡RETO!

Vamos a proponeros algo diferente a lo que hemos estado haciendo hasta ahora, pero a la vez que nos sirva perfectamente para aplicar todos los conocimientos que hemos visto sobre navegación y el uso de navbar.

Queremos que realicéis una web de una sola página donde aparezca desarrollada una noticia, un relato corto o cualquier otra historia que pueda ser divisible por capítulos:

Crea un nuevo .html
Incluye una barra de navegación fija.
Enlaza los diferentes ítems y enlaces de la barra de navegación con los diferentes capítulos del contenido:
- Al menos debe tener un título principal.

- Una imagen con un resumen del texto posterior al principio.

- Al menos 3 capítulos.
